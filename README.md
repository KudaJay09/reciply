# Reciply

Reciply is a full-stack restaurant menu management platform built to help teams organize menus, manage categories, collect customer feedback, and track changes through activity logging. It includes secure authentication, role-based permissions, and CRUD workflows for menu and category management.

## Features

- Authentication and session handling with Better Auth
- Role- and permission-based access control
- Category management
- Menu item management with availability, pricing, discounts, and images
- Customer feedback and ratings for menu items
- Activity logging for key user actions
- MongoDB data storage powered by Prisma

## Stack

- Backend: Express, Prisma, MongoDB, Better Auth
- Frontend: React Router
